## Schedule Initial Consultation

Begin by connecting with Vidiemme's team. This step ensures alignment on your digital transformation goals across industries like Retail, Pharma, or Finance.

<Steps>
  <Step title="Visit Contact Page" icon="globe">
    Navigate to the Vidiemme contact form and provide details about your business challenges.
  </Step>
  <Step title="Submit Request" icon="send">
    Include your industry, project scope, and preferred call time. Expect a response within `24 hours`.
  </Step>
  <Step title="Join Discovery Call" icon="phone">
    Participate in a `30-60 minute` video call to outline high-level objectives.
  </Step>
</Steps>

<Callout kind="tip">
  Prepare a brief overview of your current tech stack and key pain points to maximize the call's value.
</Callout>

## Onboard to Project Development

Once requirements are clear, onboard seamlessly. Vidiemme provides tools like a starter CLI for project setup.

<Steps>
  <Step title="Sign Agreement" icon="file-signature">
    Review and sign the customized project proposal.
  </Step>
  <Step title="Install Vidiemme CLI" icon="download">
    Set up the development environment.
    
    <CodeGroup tabs="npm,yarn">
      ```bash
      npm install -g @vidiemme/cli
      ```
      ```bash
      yarn global add @vidiemme/cli
      ```
    </CodeGroup>
  </Step>
  <Step title="Initialize Project" icon="code">
    Create your solution scaffold.
    
    ```bash
    vidiemme init my-project --template ai-chatbot
    cd my-project
    vidiemme dev
    ```
  </Step>
  <Step title="Collaborate Daily" icon="users">
    Join daily standups and access the shared dashboard at `https://dashboard.vidiemme.com`.
  </Step>
</Steps>
