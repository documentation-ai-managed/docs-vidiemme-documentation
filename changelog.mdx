<Update label="2024-10-20" description="v2.0.0" tags={["breaking", "feature"]}>

## New Features

- Introduced multi-language support for conversational interfaces (10+ languages).
- New Finance analytics dashboard with real-time transaction visualization.

## Breaking Changes

- Updated API authentication to use `{API_KEY}` headers instead of query params. Migrate your calls.

<CodeGroup tabs="JavaScript,Python">
```javascript
// Old
fetch('/api/transactions?key=YOUR_API_KEY')

// New
fetch('/api/transactions', {
  headers: { 'Authorization': `Bearer ${YOUR_API_KEY}` }
})
```
```python
# Old
requests.get('/api/transactions', params={'key': 'YOUR_API_KEY'})

# New
requests.get('/api/transactions', headers={'Authorization': 'Bearer YOUR_API_KEY'})
```
</CodeGroup>

## How to Check Your Version

Follow these steps to verify your installed version and apply updates.

<Steps>
  <Step title="Verify Current Version" icon="check-circle">
    Run the version check command in your project directory.

    <CodeGroup tabs="npm,yarn">
```bash
npm list vidiemme-sdk
```
```bash
yarn list vidiemme-sdk
```
    </CodeGroup>
  </Step>
  <Step title="Update SDK" icon="download">
    Install the latest version.

    ```bash
    npm install vidiemme-sdk@latest
    ```
  </Step>
  <Step title="Test Changes" icon="play-circle">
    Run your integration tests and review breaking change notes above.
  </Step>
</Steps>

<Callout kind="tip">
  Enable auto-updates in your config file to receive notifications for critical patches.
</Callout>
