## Sample AI Integration

Integrate Vidiemme's conversational AI easily.

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const vidiemmeAI = require('vidiemme-ai-sdk');

  async function chat(message) {
    const response = await vidiemmeAI.converse({
      message: message,
      apiKey: 'YOUR_API_KEY'
    });
    console.log(response.reply);
  }

  chat('Hello, recommend products');
  ```
  ```python
  from vidiemme_ai import VidiemmeAI

  ai = VidiemmeAI(api_key='YOUR_API_KEY')

  response = ai.converse(message='Hello, recommend products')
  print(response.reply)
  ```
</CodeGroup>
